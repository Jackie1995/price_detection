城市<br>

品牌名称<br>
门店ID<br>
管家<br>

联系电话<br>
页面出租面积<br>
页面价格<br>
策略价格区间-商圈下界<br>
策略价格区间-商圈上界<br>
策略价格区间-品牌下界<br>

页面价格/商圈下界<br>
页面价格/商圈上界<br>
页面价格/品牌下界<br>
页面价格/品牌上界<br>

pt<br>
